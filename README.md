# Glimpse: Enterprise-Grade Shoppable Video SDK

**Glimpse** is a high-fidelity B2B SDK that enables streaming platforms (Netflix, YouTube, Hulu, etc.) to integrate real-time product recognition and AI-powered product substitution. Brands pay to have their products "Glowing" for instant checkout, or even substituted (e.g., Coke -> Pepsi) frame-by-frame using advanced AI models.

## Project Structure
- `docs/`: In-depth documentation, roadmap, and architecture.
- `backend/`: FastAPI intelligence layer powered by Gemini Flash for vision and substitution logic.
- `sdk/`: The core Glimpse integration library for 3rd-party video players.
- `demo/netflix-clone`: A premium Netflix clone demonstrating "Deep" SDK integration.
