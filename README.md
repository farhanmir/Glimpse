# Glimpse: AI-Driven Shoppable Video SDK

Glimpse is an enterprise-grade SDK that automatically detects products in streaming videos and provides real-time shopping links.

## Project Structure
- `.docs/`: In-depth documentation, roadmap, and architecture.
- `backend/`: FastAPI intelligence layer powered by Gemini 2.0 Flash.
- `sdk/`: The core Glimpse integration library for video players.
- `demo/`: A sample platform demonstrating Glimpse integration.
